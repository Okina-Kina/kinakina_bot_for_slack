# kinakina_bot_for_slack
Executable bot on slack.
The bot is capable of simple conversation.

# Dependency
* slack_bolt

# Setup
Windows

Install Python Library
Recommendation to use venv.

```bash
$ pip install slack_bolt
```

# Usage
Windows
```bash
$ python run.py
```
